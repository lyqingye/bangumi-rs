### 像数 1, 2, 3 一样容易

```bash
# 1. 克隆项目
git clone https://github.com/lyqingye/bangumi-rs.git

# 2. 配置文件
cp config.example.toml config.toml

# 3. 使用 Docker 运行
docker-compose up -d
```
